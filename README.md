# Sistema Gestión Clientes Java

Aplicación desktop desarrollada en Java Swing para la gestión de clientes con conexión a MySQL.

## Tecnologías utilizadas

- Java SE
- Java Swing (GUI)
- MySQL
- JDBC
- Apache NetBeans
- XAMPP
- Arquitectura MVC

## Funcionalidades

- Registro de clientes  
- Listado en JTable  
- Búsqueda por nombre  
- Edición de clientes  
- Eliminación de clientes  
- Validaciones en formulario  
- Conexión a base de datos MySQL  

## Base de datos

Base de datos MySQL creada en XAMPP.

Tabla `clientes` con los siguientes campos:

- id
- nombre
- email
- telefono
- direccion
- tipo_cliente
- estado
- observaciones
- fecha_registro
